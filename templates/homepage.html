Welcome

This site is a work in progress. I add features I like to this site for practice and demonstration purposes.

    Fully functional features

  • Profiles extended off of django-userena authentication system
  • Fluid, responsive navbar and skeleton built in Bootstrap
  • Sexy jquery scroll to top icon appears only when needed
  • Javascript generates img titles from alt tags across the site
  • userena messaging system - needs serious improvements
  • Gmail email backend
  • Setup site to serve over more secure https
  • Setup two honeypots
  • Super awesome Zocial social CSS buttons
  • Created gravatar templatetag for better gravatar avatar integration
  • Created Beachmint Jewelry application

    • Future features

  • django-activity-stream for publishing social activity
  • django-social-auth integration with facebook, twitter, linkedin, google and github --Disabled--
  • Messaging System improvements
  • Fix image storage(OpenShift and built-in userena image storage aren't getting along)
